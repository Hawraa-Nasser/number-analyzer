# Number Analyzer

This is a simple C++ project called "Number Analyzer".

## Features
The program allows the user to enter a number and analyze it by:
- Determining whether the number is positive, negative, or zero
- Checking if the number is even or odd
- Calculating the square and square root

## About
This project was created as part of the System Management course to practice programming basics and teamwork using GitHub. Other students contributed by adding new features to the project.
